A simple and lightweight YouTube Video Block for WordPress that lets you easily embed YouTube videos in your content.

== Description ==

The YouTube Video Block provides a straightforward way to embed YouTube videos in your WordPress content using the block editor. 

Key Features:
* Simple YouTube video embedding
* Clean, responsive video display
* Gutenberg-native block interface
* Lightweight implementation
* Easy to use with just a YouTube URL

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/yt-video-block` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. In the block editor, search for "YouTube" or "Video" to find and add the block
4. Paste your YouTube video URL into the block

== Frequently Asked Questions ==

= How do I add a YouTube video? =

Simply add the YouTube Video Block to your post or page, then paste the YouTube video URL into the block.

= What YouTube URL formats are supported? =

The block supports standard YouTube URLs, including both full URLs and shortened versions.
